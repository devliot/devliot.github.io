  <action>
Create `scripts/build-og-pages.mjs` as a Node.js ESM script. This script performs TWO operations:

**Operation A: Compute reading time and enrich index.json**

1. Read `public/articles/index.json`
2. For each article, read `public/articles/{slug}/index.html`
3. Strip HTML tags: `html.replace(/<[^>]+>/g, ' ').replace(/\s+/g, ' ').trim()`
4. Count words: `text.split(' ').filter(w => w.length > 0).length`
5. Compute reading time: `Math.ceil(words / 238)` (238 WPM, research-backed median)
6. Inject `readingTime` into each article entry in the registry object
7. Write enriched `index.json` back to `public/articles/index.json` (so Vite copies it to dist)

**Operation B: Generate OG HTML pages into dist/**

1. For each article in the (now enriched) registry:
2. Build the absolute image URL: `https://devliot.github.io/devliot/${article.image}` (only if `article.image` is non-empty)
3. Generate a minimal HTML file with these meta tags in `<head>`:
   - `<meta property="og:type" content="article" />`
   - `<meta property="og:title" content="{escaped title}" />`
   - `<meta property="og:description" content="{escaped description}" />` (omit tag entirely if description is empty)
   - `<meta property="og:url" content="https://devliot.github.io/devliot/#/article/{slug}" />`
   - `<meta property="og:image" content="{absolute image URL}" />` (omit tag entirely if no image)
   - `<meta name="twitter:card" content="summary_large_image" />`
   - `<meta name="twitter:title" content="{escaped title}" />`
   - `<meta name="twitter:description" content="{escaped description}" />` (omit if empty)
   - `<meta name="twitter:image" content="{absolute image URL}" />` (omit if no image)
4. Add `<script>window.location.replace('/devliot/#/article/{slug}');</script>` for human redirect
5. Write to `dist/articles/{slug}/og.html`

**HTML escaping function** (required -- T-05-01 XSS prevention):
```javascript
function escapeHtml(str) {
  return str
    .replace(/&/g, '&amp;')
    .replace(/</g, '&lt;')
    .replace(/>/g, '&gt;')
    .replace(/"/g, '&quot;');
}
```

**Slug validation** (required -- T-05-02 path traversal prevention):
Before writing any file, validate each slug against `/^[a-zA-Z0-9_-]+$/`. Skip articles with invalid slugs and log a warning.

**Constants at top of file:**
```javascript
const SITE_URL = 'https://devliot.github.io';
const BASE_URL = '/devliot/';
const WPM = 238;
const SLUG_PATTERN = /^[a-zA-Z0-9_-]+$/;
```

**Critical: This script runs in TWO phases of the build pipeline:**
- Phase 1 (before vite build): `node scripts/build-og-pages.mjs --enrich` -- enriches index.json with readingTime
- Phase 2 (after vite build): `node scripts/build-og-pages.mjs --generate` -- writes OG HTML to dist/

Implement this with a CLI flag check:
```javascript
const mode = process.argv[2];
if (mode === '--enrich' || !mode) { /* Operation A */ }
if (mode === '--generate' || !mode) { /* Operation B */ }
```

When called with no flag, it runs both operations (for local testing). The build command will call each phase separately.

**Console output:** Log `Enriched index.json: {N} articles, readingTime computed` and `Generated OG pages: {N} articles in dist/`.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Author content -> OG HTML template | Article title and description (author-written strings in index.json) are injected into generated HTML files |
| Article slug -> file system path | Slug values determine output file paths for OG HTML pages |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Tampering | `scripts/build-og-pages.mjs` template injection | mitigate | HTML-escape all author-provided strings (`escapeHtml()` escaping `&`, `<`, `>`, `"`) before injecting into OG HTML template. Prevents XSS if OG pages are rendered in a browser. |
| T-05-02 | Tampering | `scripts/build-og-pages.mjs` file path construction | mitigate | Validate slug against `SLUG_PATTERN = /^[a-zA-Z0-9_-]+$/` before constructing output path. Skip and warn for invalid slugs. Prevents path traversal via crafted slug values. |
| T-05-03 | Information Disclosure | OG HTML pages expose article metadata | accept | OG pages contain only public article metadata (title, description, image URL). No private data. This is the intended purpose of these pages. |
</threat_model>
